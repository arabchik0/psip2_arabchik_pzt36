Примеры методов объекта String

Текст для анализа: "Это пример текста с некоторыми словами для обработки."

Метод String.split(): Разбиение текста на слова

    Метод String.match(): Поиск слов, начинающихся с буквы 'н'

      Метод String.search(): Поиск индекса слова "пример"

      Метод String.replace(): Замена слова "некоторыми" на "различными"
